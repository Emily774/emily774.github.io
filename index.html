My Projects

Times Lord Collectables

February 2021
Doctor Who is an iconic character who has fascinated generations of viewers. I designed a personalised website populated with existing 'TimeLord Characters' from the popular BBC sci-fi show. Character profiles are pulled from an SQL database using PHP and users can use a post request to add their own generic Time Lords to the database using the form at the bottom of the page. The new characters are then displayed on the main webpage.

Memory Game

February 2021
This fun memory game enables you to virtually match pairs of home-drawn hamster cards! A team project, this game was built using JavaScript, HTML and CSS. I focused on creating event handlers to ensure that the cards responded appropriately to being selected. No more than two cards can be selected at any one time and they flip back over after a designated period.
Git Repository: https://github.com/iO-Academy/2021-jan-pairs-game
Live Version: https://dev.io-academy.uk/projects/2021-jan/2021-jan-pairs-game/

Biscuit App
