February 2021

Memory Game

February 2021
This fun memory game enables you to virtually match pairs of home-drawn hamster cards! A team project, this game was built using JavaScript, HTML and CSS. I focused on creating event handlers to ensure that the cards responded appropriately to being selected. No more than two cards can be selected at any one time and they flip back over after a designated period.
